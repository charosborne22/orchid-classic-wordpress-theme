== Description ==
Orchid is a minimalist theme with subtle animation. Ideal for blogs, portfolios, or business websites.

== Installation ==
1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
2. Click ‘Add New’ to upload theme
3. Click on the 'Activate' button to use your new theme right away.
4. Navigate to Appearance > Customize in your admin panel and customize to your taste.
